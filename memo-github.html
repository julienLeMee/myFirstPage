Github (and Terminal) Reminders

Les commandes de bases de GIT

git init : initialise un dépôt Git.

git add : ajoute un fichier à un dépôt Git.

git commit : signale à Git que vous avez terminé de modifier un fichier.

git push : envoie les modifications de votre dépôt Git sur votre serveur.

git pull : récupère les modifications d'un dépôt Git distant.

git status : affiche les modifications non enregistrées.

git log : affiche les modifications enregistrées.

git diff : affiche les modifications enregistrées.

git diff --staged : affiche les modifications enregistrées.

Les commandes de bases du Terminal

mkdir : crée un dossier.

cd : change de dossier.

ls : liste les fichiers et dossiers d'un dossier.

rm : supprime un fichier ou un dossier.

mv : renomme un fichier ou un dossier.

cp : copie un fichier ou un dossier.

cat : affiche le contenu d'un fichier.

touch : crée un fichier.
