javaScript Reminders

Les function de JavaScript

  • alert()
    • affiche une alerte à l'écran

  • confirm()
    • affiche une alerte à l'écran avec un bouton OK et un bouton Annuler

  • prompt()
    • affiche une alerte à l'écran avec un champ de saisie

  • console.log()
    • affiche dans la console du navigateur

  • console.error()
    • affiche dans la console du navigateur en rouge

  • console.warn()
    • affiche dans la console du navigateur en orange

  • console.info()
    • affiche dans la console du navigateur en bleu

  • console.time()
    • démarre un chronomètre

  • console.timeEnd()
    • arrête un chronomètre

  • console.clear()
    • efface la console du navigateur

  • console.table()
    • affiche dans la console du navigateur un tableau

  • console.group()
    • démarre un groupe de logs

  • console.groupEnd()
    • arrête un groupe de logs

  • console.groupCollapsed()
    • démarre un groupe de logs avec un titre

  • console.count()
    • compte le nombre de fois qu'une instruction est exécutée

Les event de JavaScript

  • onclick
    • lorsque l'on clique sur un élément

  • onmouseover
    • lorsque la souris passe sur un élément

  • onmouseout
    • lorsque la souris sort de l'élément

  • onmousemove
    • lorsque la souris bouge sur un élément

  • onkeydown
    • lorsque l'on appuie sur une touche du clavier

  • onkeyup
    • lorsque l'on relâche une touche du clavier

  • onkeypress
    • lorsque l'on tape dans un champ de saisie

  • onload
    • lorsque le document est chargé

  • onunload
    • lorsque le document est déchargé

  • onchange
    • lorsque l'on change la valeur d'un champ de saisie

  • onsubmit
    • lorsque l'on soumet un formulaire

  • onreset
    • lorsque l'on réinitialise un formulaire

  • onselect
    • lorsque l'on sélectionne un élément dans un champ de saisie

  • onselectstart
    • lorsque l'on commence à sélectionner un élément dans un champ de saisie

  • onselecttext
    • lorsque l'on sélectionne un élément dans un champ de saisie

animation au scroll

const body = document.querySelector('body');

window.addEventListener('scroll', () => {
console.dir(document.documentElement.scrollTop);
if (document.documentElement.scrollTop > 800) {
document.body.classList.add('bgActive');
if (document.documentElement.scrollTop > 2000) {
document.body.classList.add('bgActive2');
if (document.documentElement.scrollTop > 3000) {
document.body.classList.add('bgActive3');
}
}
}


if (document.documentElement.scrollTop < 3000) {
document.body.classList.remove('bgActive3');
if (document.documentElement.scrollTop < 2000) {
document.body.classList.remove('bgActive2');
if (document.documentElement.scrollTop < 800) {
document.body.classList.remove('bgActive');
}
}
}
});
